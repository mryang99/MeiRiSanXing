# 每日三醒 (MeiRiSanXing)

一款将冥想计时、待办管理、每日反思融为一体的 iOS 个人成长工具。

## 功能

- 🧘 **冥想计时器** — 开始/暂停/结束计时，自动记录时长 + 备注，支持手动补录
- ✅ **待办任务** — 紧急程度标记（🔴紧急/🔵普通/🟢低优）、截止日期、进度条、完成勾选
- 📝 **每日反思** — 结构化模板 + 自由段落混合编辑，情绪标记
- 📊 **每周/每月总结** — 自动汇总冥想时长、任务完成率、反思天数

## 技术栈

- Swift 6 + SwiftUI
- SwiftData (本地持久化)
- iOS 18+
- UserNotifications (本地通知)
- XcodeGen (项目生成)
